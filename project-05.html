05 / University Registration System

Spring 2022
Manue Alaimo ⏤ 2024
Linkedin
Github
Tech Stack

Python, Flask, SQL, AWS Server, HTML, CSS, JS

Hosted in the AWS Cloud, this web application and supporting database is designed to meet a university’s course registration needs.

Students applications and registration to the University include form validation. Security enhancements authenticate a user’s credential to access a student’s information.

User View - Faculty

 Allows a faculty member to view their students. In this example, these are students the faculty member advises. The faculty member can view different types of information relevant to the student.

User View - Form Example

 Allows any user to change their personal information which updates the database upon submission.

User View - Graduate Secretary

 Allows the Graduate Secretary to view all applications to the University.
