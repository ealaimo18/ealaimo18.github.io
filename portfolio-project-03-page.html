03 / RL for Building Emissions

Fall 2023

City Learn Challenge 2023 ⏤ Multi-agent reinforcement learning on building controls to reduce greenhouse gas emissions and energy consumption.

 

Developed a new Soft-Actor Critic algorithm for the Agent to learn both on a value and policy based reward system in the environment.

 

Tech Stack

Python, Stable Baseline Package, CityLearn Package

 

 

Trained Soft Actor Critic Results

 

The upward trending SAC reward results show that the Agent was able to successfully learn with the reward system implemented.

 

District Level Performance Indicators

 

The building efficiency baseline is 1.0, with the objective to approach 0.

 

The SAC deployed improved cost, carbon emissions, electric consumption, and the average electric peak.

 

To achieve this, the SAC was provided with future weather predictions and prioritized a higher state of charge remaining in reserve batteries.

 

Building Level Performance Indicators

 

 

Individual buildings in the district were able to optimize significantly below the average.

 

Here the blue bar indicates the SAC’s model to reduce the baseline.

 

Each training model was run under a minimum of 45 episodes to provide an ample timeline.
