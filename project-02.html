02 / Air Quality Monitoring System

Spring 2023
Manue Alaimo ⏤ 2024
Linkedin
Github
Tech Stack

Python, Flask, NGINX, Google Maps API, SQL
HTML, CSS, JS

An IOT device to inform the public of unsafe air quality conditions ⏤ aims increase accuracy of air quality information by placing sensors directly in a user’s environment.

A system of Raspberry Pi nodes collect and send air quality data to a central server node to deliver push notifications and display a user interface.

Raspberry Pi 4, SDS011 Air Quality Sensor
