04 / Wizdom Wizard

Spring 2023

An Android application to share tokens of advice ⏤ leverages Advice Slip Generator API. Implements Firebase for user account creation and verification and stores user’s favorites in Firebase Realtime database.

 

Integrated with localization to allow for language translation.

Tech Stack

Kotlin, Firebase Realtime Database, Firebase Authentication, Advice Slip Generator API
